# 遊戲發酵室網站圖片建議與生成 Prompt

這份文件整理了目前首頁中所有適合放圖片的地方，並為每個位置提供建議的生成 prompt。

整體共用風格關鍵字：

- 淺藍色
- 低彩度
- 輕盈空氣感
- 微玻璃感
- 柔和發光
- 泡泡與發酵感
- 創意實驗室
- 議題型遊戲品牌
- 現代、乾淨、高質感
- 插畫感，但不要太幼稚

建議全站圖片共通限制：

- 不要太鮮豔
- 不要太卡通童書感
- 不要賽博龐克
- 不要厚重深色背景
- 不要雜亂複雜構圖
- 保留留白
- 優先使用橫式構圖
- 盡量避免文字出現在圖片內

---

## 1. Hero 主視覺插圖

位置：

- 首頁 Hero 右側主視覺

用途：

- 作為品牌第一眼的代表圖
- 建立「遊戲 + 發酵 + 社群 + 實驗室」的世界觀

建議尺寸：

- 1600 x 1200
- 或 4:3 橫式

建議 prompt：

```text
A premium soft illustration for the official website of "Game Fermentation Lab", a creative studio making issue-based social games. A translucent white jelly slime mascot floating in a light blue airy laboratory scene, surrounded by bubbles, soft glow, tiny charts, magnifying glass, chat bubbles, potion flask, playful interface fragments, and social interaction symbols. Low saturation pastel blue palette, clean composition, glassmorphism feeling, elegant negative space, modern editorial brand style, lightweight atmosphere, subtle experimental lab mood, soft highlights, polished website hero artwork, no text, no dark background, no childish cartoon style.
```

## 2. Hero 輔助小圖示素材

- 對話框
- 藥水瓶

- 漂浮泡泡

建議 prompt：

```text
A set of soft minimal floating object illustrations for a light blue brand website: magnifying glass, chart panel, speech bubble, potion bottle, tiny sword, burger icon, glossy bubbles. Low saturation pastel blue and white palette, semi-transparent jelly-like material, subtle glow, clean modern editorial illustration, transparent background, high quality, gentle and playful but not childish.
```
